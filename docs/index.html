Install

Download a release or build from source.

Use GitHub Releases for packaged builds, or run locally with the .NET 10 SDK.

Release builds

Grab the latest desktop package from GitHub Releases. On macOS, remove the quarantine flag if Gatekeeper blocks the app.

Open Releases 
xattr -cr "/Applications/BusLane.app"

Build from source

Clone the repository, restore the .NET project, and run the Avalonia desktop app locally.

git clone https://github.com/soliktomasz/BusLane.git
cd BusLane
dotnet build
dotnet run --project BusLane/BusLane.csproj
FAQ

Common questions.

Who is BusLane for?

BusLane is for engineers and operators who need a focused desktop workflow for Azure Service Bus debugging, message inspection, and daily queue operations.

Which platforms are supported?

BusLane is built with Avalonia and targets macOS, Windows, and Linux.

How does BusLane handle sensitive connection data?

Connection strings are stored locally with encryption, and the app also supports an optional launch-time app lock.
